# Barteguiden-App
A React Native app for Barteguiden.

## Setup
You'll need `Node.js 4.0 or newer`, `watchman` is also recommended. 
For the iOS app, you'll also need OSX and Xcode.

Run `npm install -g react-native-cli`

More information on getting started can be found [here](https://facebook.github.io/react-native/docs/getting-started.html).

## Run the apps
### iOS
- Open `ios/BarteguidenApp.xcodeproj` and hit run in Xcode or run `react-native run-ios` in the terminal
- Hit ⌘-R in the iOS simulator to reload and see changes

### Android
We recommend using [Genymotion](https://www.genymotion.com/) for device simulation. 
- Start your simulator
- Run `react-native run-android`
- Press the menu button (F2 by default, or ⌘+M in Genymotion) and select Reload JS to see your changes

If you wish to run the app on a physical device, read [here](https://facebook.github.io/react-native/docs/running-on-device.html).
